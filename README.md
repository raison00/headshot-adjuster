
# Run and deploy app

This contains everything to run app locally.

View app: https://ai.studio/apps/drive/1MjFSAjRQrkkFf6045SVhnwGSVB7WcN3j

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
